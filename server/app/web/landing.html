三条差异化，撑起「随行」两个字

零痕迹、即问即得、懂你的文件——三者缺一不可。

01

零痕迹

公司电脑默认不下载文件，只在线预览，关页即失。离职一键吊销令牌，服务端即刻切断访问，如同从未存在。

  • 浏览器默认禁下载，需验证密码才开启临时窗口
  • 在线预览强制 no-store，不进磁盘缓存
  • HTML / SVG 等可执行类型禁止浏览器预览
  • 吊销令牌 + 密码版本号 bump，旧凭证立即失效
02

即问即得

不用翻文件夹，用自然语言告诉 Agent 意图，它找到文件、传好、通知你。

  • 语义检索：搜"上季度报价"按内容匹配
  • AI 问答：问"尾款什么时候付"，Agent 在文件里找答案
  • 支持 Word / Excel / PDF / PPT 内容提取
  • 语义搜索失败自动回退关键词搜索，不掉链子
03

懂你的文件

Agent 索引过文件名和内容，能分类、能提醒、能建议。

  • 智能同步建议：基于修改时间推断该推送到哪
  • 离职清理助手：设备令牌审计 + 敏感文件清单
  • 清理建议：找出长期未用的文件
  • 笔记双向链接：[[wiki link]] 语法串联知识

三种人，一种用法

家里端、公司端、管理员——三端不对称，各取所需。

家里

装软件、完全拥有

轻量守护进程监听文件夹，新增 / 修改 / 删除自动双向同步到服务器。冲突检测 + .conflict 副本，绝不静默覆盖。

公司

零安装、只看不留

打开浏览器，登录域名找文件 / 对话 / 在线预览。默认禁止下载（需临时窗口），没有客户端、配置文件或同步缓存。

管理员

独立后台、全量管控

用户 CRUD、配额 / 状态管理、设备令牌吊销、大模型配置、系统设置运行时调整、审计日志全量记录。

安全是 DNA，不是附加功能

零痕迹、三重隔离、PII 服务端脱敏、全量审计——每一层都不可或缺。

一键部署，长在你自己的服务器上

一行命令，自动下载配置、生成密钥、拉镜像启动。无需 clone 源码。

1

一键脚本

curl -fsSL https://raw.githubusercontent.com/wupenghello/Suixingdang/main/install.sh | bash

脚本交互式收集域名与密码，自动生成三把独立密钥，写 .env（权限 600），拉镜像启动。

2

登录配置

# 打开 https://你的域名/admin
# 账号 admin（密码见 .env）
# 在「大模型配置」页添加 DeepSeek / OpenAI API Key

管理员在后台添加大模型并分配给用户，即可使用 AI 对话、语义检索、内容摘要等能力。

3

# 家里守护进程：创建设备令牌后配置
# 公司端：浏览器打开即用，零安装
