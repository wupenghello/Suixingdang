01

零痕迹

公司电脑默认不下载文件，只在线预览，关页即失。离职一键吊销令牌，服务端即刻切断访问，如同从未存在。

  • 浏览器默认禁下载，需验证密码才开启临时窗口
  • 在线预览强制 no-store，不进磁盘缓存
  • HTML / SVG 等可执行类型禁止浏览器预览
  • 吊销令牌 + 密码版本号 bump，旧凭证立即失效
02

即问即得

不用翻文件夹，用自然语言告诉 Agent 意图，它找到文件、传好、通知你。

  • 语义检索：搜"上季度报价"按内容匹配
  • AI 问答：问"尾款什么时候付"，Agent 在文件里找答案
  • 支持 Word / Excel / PDF / PPT 内容提取
  • 语义搜索失败自动回退关键词搜索，不掉链子
03
